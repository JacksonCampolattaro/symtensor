The index sequence is converted to a flat index at compile-time, allowing for performance equivalent to hand-written member access. mytensor.at<X, X, X>() should be equivalent to mytensor.flat()[0].

Direct access to underlying data structure.

Returns a reference to the underlying flat storage.

Note

No guarantees are provided about the definition and layout of the flat data structure, except for the fact that it is an iterable with a size(), begin() and end() method. The elements of the symmetric tensor may be rearranged in order to facilitate vectorized operations, for the best possible performance.
